MusicXML
=========

Criação de arquivo XML baseado nas informações de ID3 lidas em arquivos MP3.
O modelo do XML de saída segue o padrão abaixo:

    <?xml version="1.0" encoding="UTF-8" standalone="no"?>
    <library>
        <artist>
            <name>AUTUMN TEARS</name>
            <album>
                <name>ABSOLUTION</name>
                <music>
                    <tiltle>THE ABSOLUTION OF WHAT ONCE WAS</tiltle>
                </music>
                <music>
                    <tiltle>THE NEVER</tiltle>
                </music>
                <music>
                    <tiltle>THE DANCE</tiltle>
                </music>
            </album>
        </artist>
    </library>
